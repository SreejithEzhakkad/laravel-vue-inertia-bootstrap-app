## Installation

Clone the repository.

`$ git clone git@github.com:SreejithEzhakkad/laravel-vue-inertia-bootstrap-app.git`

  Create a MySQL database. Copy `.env.example` to `.env` and change values.
Do the following

`$ composer install`

`$ npm install`

`$ npm run production`

`$ php artisan migrate`

`$ php artisan serve`
